개인적으로 중요하다고 느낀 내용과 실습 과정에서 배운 점들을 정리했습니다.

- CHAPTER 01 대시보드 이해
- CHAPTER 02 정확한 대시보드 구성에 필수적인 함수와 수식
- CHAPTER 03 보고서에 필요한 데이터 시각화① : 조건부 서식
- CHAPTER 04 보고서에 필요한 데이터 시각화② : 차트
- CHAPTER 05 피벗 테이블 보고서 활용
- CHAPTER 06 대시보드 구성
