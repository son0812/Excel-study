## CH1 핵심 필기 내용

- 엑셀 표 등록 방법 : `Ctrl` + `T`
-  `Ctrl` + `T`로 실행한 [표 만들기] 대화상자에는 [머리글 포함] 옵션 有 → (해당 표 범위에 컬럼명이 입력되어있으면 반드시 이 옵션 선택)
- 그냥 표형태로는 다른 표를 취합했을 때 차트가 업데이트 되지 않는데,  `Ctrl` + `T`를 통해 표로 만들면 다른 표를 취합했을 때 차트가 업데이트 됨
- [테이블 디자인] > [슬라이서 삽입] > 이 과정을 통해 원하는 지점만 선택하여 차트에 반영 가능
+ [슬라이서]는 엑셀 표와 피벗 테이블에서만 사용 가능
