## CH2 핵심 필기 내용

### SECTION01 IF 함수로 대표되는 판단 함수
-`IF(조건식, TRUE일 때 값, FALSE일 때 값)` : IF함수는 한번에 하나의 조건만 처리하기에 여러 조건을 처리하고 싶으면 IF문 안에 IF문을 중첩해서 넣어야함
- `IFS(조건식1, TRUE일 떄 반환1, 조건식2, TRUE일 때 반환2...)` : IF함수를 여러번 중첩해야하는 불편함 해결하고자 여러개의 조건을 판단하는 함수
- `AND(조건1,조건2,조건3..)` : 모든 조건이 TRUE일 때 TRUE만환하고 하나라도 FALSE면 FALSE반환
- `OR (조건1,조건2,조건3..)` : 하나라도 TRUE면 TRUE반환, 모두 FALSE면 FALSE반환
- IFS 함수만으로 처리하기 어려운 다중조건의 경우 보통 AND,OR과 같은 논리 함수를 함께 사용
- `IFERROR(계산식, 에러가 발생할 때 반환할 값)` : 수식 에러를 사용자가 원하는 값으로 변경할 수 있는 함수
※수식 에러 : #N/A, #DIV/0!, #VALUE!, #NAME? 등


 📍핵심 NOTE
 
성적이 90점 이상일 경우 학점을  A, 80점대 B, 70점대 C, 60점대 D, 60점 미만 F로 변환하라는 문제에서 `=IFS($G3>=90,"A",$G3>=80,"B",$G3>=70,"C",$G3>=60,"D",$G3<60,"F")`를 사용하여 풀이했다.

그러나 마지막 조건에서 굳이 `$G3<60`으로 설정하지 않고 TRUE를 조건으로 넣으면 앞의 조건에 해당하지 않는 나머지 모든 경우를 처리해준다.

따라서 `=IFS($G3>=90,"A",$G3>=80,"B",$G3>=70,"C",$G3>=60,"D",TRUE,"F")`가 더 효율적인 방식이다. 

- `=IF(AND($G3>=5,$H3>=250), "지급","")`를 AND 조건절 없이 사용하려면 `=IF($G3>=5, IF($H3>=250,"지급",""),"")`으로도 가능하다

- `=IF(OR($G3<=50,$G3/$H3<=3),"재주문","")`를 OR조건절 없이 사용하려면 `=IF($G3<=50,"재주문",IF($G3/$H3<=3,"재주문",""))`으로도 가능하다.
