### SECTION01 IF 함수로 대표되는 판단 함수
-`IF(조건식, TRUE일 때 값, FALSE일 때 값)` : IF함수는 한번에 하나의 조건만 처리하기에 여러 조건을 처리하고 싶으면 IF문 안에 IF문을 중첩해서 넣어야함
- `IFS(조건식1, TRUE일 떄 반환1, 조건식2, TRUE일 때 반환2...)` : IF함수를 여러번 중첩해야하는 불편함 해결하고자 여러개의 조건을 판단하는 함수
- `AND(조건1,조건2,조건3..)` : 모든 조건이 TRUE일 때 TRUE만환하고 하나라도 FALSE면 FALSE반환
- `OR (조건1,조건2,조건3..)` : 하나라도 TRUE면 TRUE반환, 모두 FALSE면 FALSE반환
- IFS 함수만으로 처리하기 어려운 다중조건의 경우 보통 AND,OR과 같은 논리 함수를 함께 사용
- `IFERROR(계산식, 에러가 발생할 때 반환할 값)` : 수식 에러를 사용자가 원하는 값으로 변경할 수 있는 함수
※수식 에러 : #N/A, #DIV/0!, #VALUE!, #NAME? 등

📍핵심 NOTE
나같은 경우, 성적이 90점 이상일 경우 학점을  A, 80점대 B, 70점대 C, 60점대 D, 60점 미만 F로 변환하라는 문제에서 `=IFS($G3>=90,"A",$G3>=80,"B",$G3>=70,"C",$G3>=60,"D",$G3<60,"F")`를 사용하였다.
그러나 마지막 조건에서 굳이 `$G3<60`으로 설정하지 않고 TRUE를 조건으로 넣으면 앞의 조건에 해당하지 않는 나머지 모든 경우를 처리해준다.
따라서 `=IFS($G3>=90,"A",$G3>=80,"B",$G3>=70,"C",$G3>=60,"D",TRUE,"F")`가 더 효율적인 방식이다. 

- `=IF(AND($G3>=5,$H3>=250), "지급","")`를 AND 조건절 없이 사용하려면 `=IF($G3>=5, IF($H3>=250,"지급",""),"")`으로도 가능하다

- `=IF(OR($G3<=50,$G3/$H3<=3),"재주문","")`를 OR조건절 없이 사용하려면 `=IF($G3<=50,"재주문",IF($G3/$H3<=3,"재주문",""))`으로도 가능하다.

### SECTION02  VLOOKUP 함수
- VLOOKUP : VERTICAL + LOOKUP 수직(세로) 방향으로 값을 찾음
- `VLOOKUP(기준셀, 참조범위, 호출할 열 번호, 일치옵션)` : 일치옵션은 TRUE[1](유사일치), FALSE[0](정확히 일치)
- 📍내가 이해한 내용 : VLOOKUP(현재테이블에서 기준이 되는 셀, 참고할 다른 테이블 범위, 다른 테이블에서 몇번째 열이 이 빈 셀에 들어왔으면 좋겠는지, 정확히 일치하는지 유무)

※`#N/A`같은 수식 에러가 발생할 경우 : `IFERROR`나 `XLOOKUP`을 써서 해결 

→ `IFERROR(VLOOKUP(기준셀,참조범위, 호출 열번호, 일치옵션),"찾는 값 없음")`

→ ⭐`XLOOKUP(기준셀, 기준셀의 참조범위, 채워넣을 셀의 참조범위,"에러발생시 반환할 문구나 값")`

+ 만약 참조할 값이 오른쪽 열이 아니라 왼쪽 열에 있다면 VLOOKUP은 사용하지 못하고 INDEX, MATCH 함수를 사용해야 함

- HLOOKUP : HORIZONTAL + LOOKUP 수평(가로) 방향으로 값을 찾음
- `HLOOKUP(기준셀, 참조범위, 호출할 행 번호, 일치옵션)  : 일치옵션은 TRUE[1](유사일치), FALSE[0](정확히 일치)
-⭐HLOOKUP에서 행 번호 찾을 때 머리글(컬럼명)이 행번호1번이라는 것을 잊지말 것

- VLOOKUP, HLOOKUP 함수의 경우 각각 하나의 열, 행에서만 값을 찾을 수 있는데 INDEX, MATCH 함수를 쓰면 특정 열과 행을 동시에 지정하여 두 위치가 교차하는 값을 참조 가능
- INDEX(표 범위, 행번호, 열번호) : 다른 표에서 지정한 행/열 좌표의 값 참조 가능
- MATCH(찾을값, 찾을 범위, [1,0,-1]) : 값을 찾아 찾는 값이 몇 번째에 있는지 반환하는 함수 [1은 디폴트이기에 생략 가능, 1로 설정할 경우 찾을값보다 작은 값 중에서 가장 유사한 값/0으로 설정한 경우 정확히 일치/-1은 찾을값보다 큰 값 중에서 가장 유사한 값]

- `XLOOKUP(찾을값, 찾을 범위, 참조 범위, #N/A일 경우 대체, 찾는방법, 찾는방향) : `찾을 범위`에서 원하는 값의 위치를 찾고 `참조 범위`내 동일한 위치에 있는 값을 반환
- 찾는방법 : 디폴트는0
| 옵션 | 설명 |
|------|------|
| 1 | 정확하게 일치하거나, 다음으로 큰 값을 찾습니다. |
| 0 | 정확하게 일치하는 값을 찾습니다. |
| -1 | 정확하게 일치하거나, 다음으로 작은 값을 찾습니다. |
| 2 | 와일드카드 문자(*, ?, ~)를 사용해 값을 찾습니다. |

- 찾는 방향 : 디폴트는1
| 옵션 | 설명 |
|------|------|
| 1 | [찾을 범위] 내 첫 번째 항목부터 값을 찾습니다. |
| -1 | [찾을 범위] 내 마지막 항목부터 역순으로 찾습니다. |
| 2 | 오름차순으로 정렬된 범위에서 이진 검색으로 찾습니다. |
| -2 | 내림차순으로 정렬된 범위에서 이진 검색으로 찾습니다. |

- `XMATCH(찾을값, 찾을 범위, 찾는 방법, 찾는 방향)` : 찾을 범위에서 원하는 값을 찾아 해당 값의 위치가 찾을 범위 내 몇 번째에 있는지 위치를 파악하여 반환
- 찾는 방법 : 디폴트는 0
| 옵션 | 설명 |
|------|------|
| 1 | 정확하게 일치하거나, 다음으로 큰 값을 찾습니다. |
| 0 | 정확하게 일치하는 값을 찾습니다. |
| -1 | 정확하게 일치하거나, 다음으로 작은 값을 찾습니다. |
| 2 | 와일드카드 문자(*, ?, ~)를 사용해 값을 찾습니다. |

- 찾는 방향 : 디폴트는 1
| 옵션 | 설명 |
|------|------|
| 1 | [찾을 범위] 내 첫 번째 항목부터 찾습니다. |
| -1 | [찾을 범위] 내 마지막 항목부터 역순으로 찾습니다. |

### SECTION03 시계열 데이터를 핸들링하기 위한 날짜/시간 함수
| 날짜 단위 | 수식 | 반환 |
| :---: | :--- | :---: |
| 연도 | `=YEAR(A1) & "년"` | 2024년 |
| | `=TEXT(A1, "yy년")` | 24년 |
| 반기 | `=IF(MONTH(A1)<7, "상반기", "하반기")` | 상반기/하반기 |
| 분기 | `=ROUNDUP(MONTH(A1)/3, 0) & "분기"` | 1분기~4분기 |
| | `="Q" & ROUNDUP(MONTH(A1)/3, 0)` | Q1~Q4 |
| 월 | `=MONTH(A1) & "월"` | 1월~12월 |
| | `=TEXT(A1, "mm월")` | 01월~12월 |
| 주 | `=WEEKNUM(A1) & "주"` | 1주~54주 |
| | `=INT((DAY(A1)-WEEKDAY(A1)+13)/7) & "주"` | 1주~5주 |
| 요일 | `=TEXT(A1, "aaa")` | 월~일 |
